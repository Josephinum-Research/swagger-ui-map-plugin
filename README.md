# SwaggerUI Map Plugin
Add a map view to supported results in the SwaggerUI.

# Usage
```html
<script src="https://cdn.jsdelivr.net/npm/ol@v10.1.0/dist/ol.js" crossorigin></script>
<script src="swagger-ui-map-plugin.js"></script>
```

```javascript
SwaggerUIBundle({
    ...
    plugins: [
        SwaggerUiMapPlugin
    ],
    ...
});
```

# Dev
`npm install`
`npm run dev`
`npm run build`
